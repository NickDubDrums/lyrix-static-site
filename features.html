Features

Lyrix Bridge (JUCE Plugin)

Sends events to the web server; Connect button + QR.

Lyrix Host (mini-server)

Quick start, QR and instant distribution on the local network.

Web App

Responsive UI, split Lyrics/Chords, behavior badges, preferences.

Universal MIDI Controller

Note 97 = START, 96 = END; CC20/52 (14-bit) for Lyrics, CC21/53 for Chords (+22 root, +23 quality).
